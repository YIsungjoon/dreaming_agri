# AI

가상데이터 생성, 일정 예측, 문서 기반 질의응답, 제한된 이미지 상태 확인과 평가 코드를 관리하는 독립 영역입니다.

```text
ai/
├─ src/
│  ├─ synthetic-data/
│  ├─ forecasting/
│  ├─ rag/
│  └─ vision/
├─ tests/
└─ evaluations/
```

현재는 AI 배포 환경과 모델이 확정되지 않았으므로 특정 웹 프레임워크나 모델 SDK를 설치하지 않습니다. 구현 시 다음 계약을 우선 정의합니다.

- 입력: 농장·작기 식별자, 시간 범위, 환경·생육·작업 데이터 참조
- 출력: 제안, 예상 범위, 신뢰 수준, 사용 근거, 추가 확인 행동
- 금지: 자동제어 확정, 근거 없는 날짜 단정, 전문가 진단을 대체하는 표현
- 보안: AI 제공자 키는 브라우저에 노출하지 않고 서버 런타임에만 저장
